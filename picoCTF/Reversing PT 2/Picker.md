## Descripción
- This service can provide you with a random number, but can it do anything else?

Connect to the program with netcat:

`$ nc saturn.picoctf.net 59410`

The program's source code can be downloaded [here](https://artifacts.picoctf.net/c/513/picker-I.py).
## Solución
- La solución de este reto fue poner la palabra win, esto lo mostraba en el código y después de esto se tiene que decodificar el encriptado de hex
## Notas
- Fue uno de los retos faciles, no fue tan complicado pero no me pareció divertido
